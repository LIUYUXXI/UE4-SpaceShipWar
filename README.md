# UE4-SpaceShipWar
UE4制作的2D飞机对战小游戏
* 1、实现了飞机的移动控制和开火控制；
* 2、实现了敌人的自动生成和自动追踪玩家功能；
* 3、实现了子弹的生成和销毁功能;
* 4、实现了击杀得分统计
